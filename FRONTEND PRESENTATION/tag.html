HTML ELEMENTS

HTML element is a component of HTML document that tells a web browser how to structure and interpret a part of the HTML document.

An element in HTML usually consist of a start tag , close tag and content inserted between them. Technically, an element is a collection of start tag, attributes, end tag, content between them.

HTML element is often — but not always — created by opening and closing HTML tags, which wrap around a piece of content. Below is an illustration that labels each of the parts of an HTML element:

Most HTML elements consist of three parts:

1. The opening tag (or start tag): This marks where the element’s content begins

2.The closing tag (or end tag): This marks the end of the element’s content. The closing tag is identical to the opening tag with the addition of a forward slash (/) after the first angle bracket.

3. The content: This is placed between the element’s opening and closing tags

VOID ELEMENT

Void element: All the elements in HTML do not require to have start tag and end tag, some elements does not have content and end tag such elements are known as Void elements or empty elements. These elements are also called as unpaired tag.

Self-closing tags in HTML are sometimes also known as empty tags, void tags, singletons tags, etc. This means that these tags have no content and cannot have any children.

CATERGORIES OF HTML ELEMENTS

For the default display and styling purpose in HTML, all the elements are divided into two categories:

  1. Block-level element
  2. Inline element

Block-level element A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element. A block-level element always takes up the full width available (stretches out to the left and right as far as it can).


Following are the block-level elements in HTML.
< address>, < article>, < aside>, < blockquote>, < dd>, < div>, < figure>, < footer>, < form>, < h1>-< h6>, < header>, < hr>, < li>, < main>, < nav>, < ol>, < output>, < p>, < pre>, < section>, < table>, < tfoot>, < ul>, < video> etc.



Inline elements are those elements, which differentiate the part of a given text and provide it a particular function.

These elements does not start with new line and take width as per requirement. The Inline elements are mostly used with other elements. An inline element only takes up as much width as necessary.

EXAMPLE:

< a>, < b>, < big>, < br>, < button>, < cite>, < em>, < i>, < img>, < input>, < label>, < select>, < small>, < span>, < strong>, < sub>, < sup> , < textarea> etc.



HTML Formatting Elements

Formatting elements were designed to display special types of text. Most inline are formatting elements.

Examples:

< b> - Bold text

< strong> - Important text

< i> - Italic text

< em> - Emphasized text

< mark> - Marked text

< small> - Smaller text

< del> - Deleted text

< sub> - Subscript text

< sup> - Superscript text

Basic Elements

1. < html>: It is the root element and it defines the whole HTML document.

2. < body>: It defines the document's body.

3. < p>: It defines the paragraph.

4. < h1>: It defines a heading. Options range from h1 to h6

5. < br>: It defines a line break.

6. < hr>: It defines a horizontal rule.

7. < head>: It contains metadata about the html tags.

8. < img>: It insert/fix an image on the page.

9. < video>: It insert a media player on .

10. < a>: It creates a hyperlink from the included text. A hyperlink is an element in an HTML document that links to other document.

11. < ol>: It creates an ordered list, usually a numbered list by default.

12. < button>: It creates an interative button.

13. < header>: It denotes the header area of a page.

14. < footer>: It denotes the footer area of a page.

15. < nav>: It denotes the navigational area area of a page.

16. < div>: It defines a division or a section in an HTML document.

17. < span>: It is an inline container for phrasing content. It can be used to group elements for styling purposes .

18. < link>: It signals that the HTML document uses another document. Most commonly used to link to an external CSS file.

19. < section>: It defines the section of documents such as chapters, headers, footers or any other sections.

20. < form>: It used to create an HTML form for user input.

21. < input>: It defines the field where the user can enter data.

22. < table>: It is a representation of data arranged in rows and columns.
